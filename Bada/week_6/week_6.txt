<변수>
var : 함수레벨 / 재선언 가능 / 재할당 가능
let : 블록레벨 / 재선언 불가능 / 재할당 가능
const : 블록레벨 / 재선언 불가능 / 재할당 불가능

<배열 접근 방법>
1. index로 접근
2. for..of로 접근
3. forEach로 접근

<객체>
key와 value로 구성된 프로퍼티의 집합
1. 점 표기법
2. 대괄호 표기법

<객체 접근 방법>
for..in으로 접근 가능

<함수>
1. 함수 선언식
2. 함수 표현식
3. 화살표 함수

<DOM>
HTML 문서의 프로그래밍 interface
프로그래밍 언어가 해당 문서에 접근하여 읽고 조작
document가 root에 있음
- getElementsByTagName(태그이름)
- getElementById(아이디)
- getElementsByClassName(클래스이름)
